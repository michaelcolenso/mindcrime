# Distribution Setup Guide: Listing The Client Machine on Gumroad

## Why Gumroad

Gumroad is the best platform for this product because:
- No monthly fees (they take 10% per transaction)
- Instant setup — you can be live in under 30 minutes
- Built-in file delivery (customer gets download link after purchase)
- Built-in email collection for future marketing
- Simple analytics dashboard
- Supports PDF + multiple file delivery
- Handles payments, taxes, and refunds
- Discount codes for launch promotions

## Step-by-Step Setup

### Step 1: Create Your Gumroad Account

1. Go to **gumroad.com** and click "Start Selling"
2. Sign up with email or Google account
3. Complete your profile:
   - Display name: Your name or brand name
   - Bio: "I help freelancers build repeatable client acquisition systems using AI."
   - Profile image: Professional headshot or brand logo
4. Connect your payment method:
   - Go to Settings → Payouts
   - Connect Stripe (for credit card payments) or PayPal
   - Set your payout schedule (weekly recommended)

### Step 2: Create the Product Listing

1. Click **"New Product"** from your dashboard
2. Select **"Digital Product"**
3. Fill in the product details:

**Product Name:**
```
The Client Machine: The AI-Powered Client Acquisition System for Freelancers
```

**Price:** $39

**Short Description (for Gumroad Discover):**
```
A complete, step-by-step system for finding, reaching, and closing freelance clients — powered by AI. Includes a 13-chapter guide, 50+ AI prompts, 12 cold email templates, proposal template, discovery call script, and a 30-day implementation plan. Stop chasing clients. Build a machine that brings them to you.
```

**Cover Image:**
- Create a simple, professional product mockup using Canva (free)
- Recommended: Use Canva's "eBook Cover" template
- Design: Clean, bold typography. Dark background with accent color.
- Include the product title, subtitle, and a simple geometric/abstract visual
- Size: 1280 x 720px (Gumroad recommended)

**Product Content (files to upload):**
Upload all files as a single ZIP or as individual files:
1. `The-Client-Machine-Complete-Guide.pdf` (the main guide)
2. `AI-Prompt-Library.pdf`
3. `Cold-Email-Swipe-File.pdf`
4. `Proposal-Template.pdf` (or .md / .docx for editability)
5. `Discovery-Call-Script.pdf`
6. `Client-Onboarding-Checklist.pdf`
7. `Weekly-Pipeline-Tracker.pdf` (or .md / .xlsx for editability)

**Tip:** Upload both PDF versions (for reading) and Markdown/DOCX versions (for editing) of the templates. Customers appreciate having editable formats.

### Step 3: Configure Product Settings

1. **URL slug:** Set to `the-client-machine`
   - Your product URL will be: `yourname.gumroad.com/l/the-client-machine`

2. **Pricing:**
   - Set base price: $39
   - Enable "Allow customers to pay more than the listed price" (some will!)
   - Optional: Create a launch discount code (see Step 5)

3. **Product description (long form):**
   Copy the sales page content from `sales-page.md` into the product description. Gumroad supports Markdown formatting.

4. **Ratings & Reviews:** Enable customer ratings (social proof builds over time)

5. **Refund policy:** Set to 30-day refund window

6. **Variants (optional):**
   You can create tiers later if desired:
   - Base ($39): The complete guide + all templates
   - Premium ($79): Everything + a video walkthrough or 1-on-1 call

### Step 4: Set Up Email Collection

1. Go to **Audience** in your Gumroad dashboard
2. Gumroad automatically collects buyer emails
3. Set up a **post-purchase email** (Gumroad calls this a "Ping"):
   - Go to product settings → "After purchase"
   - Add a custom message:

```
Thanks for buying The Client Machine! 🎯

Here's how to get the most out of your purchase:

1. Start with the main guide — read Chapter 1-3 first to build your foundation
2. Use the 30-Day Launch Plan (Chapter 13) as your implementation roadmap
3. Keep the AI Prompt Library open as a reference while you work through each stage

Quick win: Open Chapter 2, run the first AI prompt to define your Ideal Client Profile, and you'll have a clear target in 15 minutes.

If you have questions or want to share a win, reply to this email. I read every message.

To your pipeline,
[Your Name]
```

### Step 5: Create a Launch Discount Code

1. Go to product settings → Discount codes
2. Create code: `LAUNCH` for 25% off ($29.25 instead of $39)
3. Set an expiration: 7 days from launch
4. Use this code in your launch posts to create urgency

### Step 6: Test the Purchase Flow

1. Create a 100% off test discount code
2. Go through the full purchase flow yourself
3. Verify:
   - Checkout page looks right
   - Payment processes (even at $0)
   - Files download correctly
   - Post-purchase email sends
   - All PDFs open properly
4. Delete the test discount code after verification

### Step 7: Publish

1. Click **"Publish"** on your product page
2. Your product is now live and purchasable
3. Share the direct link: `yourname.gumroad.com/l/the-client-machine`

## Post-Launch Optimization

### Week 1
- Monitor Gumroad analytics daily
- Respond to any customer emails within 24 hours
- Collect and address any feedback about the product

### Week 2-4
- Ask early buyers for reviews/ratings on Gumroad
- Use buyer testimonials in updated product description
- Consider a "last chance" reminder for the launch discount

### Month 2+
- Add the product to Gumroad Discover for organic traffic
- Consider creating a $9 "lite" version (just the prompt library) as a funnel
- Build an email list from buyers for future products
- Test price increases ($49, $59) as you add reviews and testimonials

## Alternative Platforms

If Gumroad doesn't suit you, these are viable alternatives:

| Platform | Fee | Best For |
|----------|-----|----------|
| **Gumroad** | 10% per sale | Simplest setup, good discovery |
| **Lemon Squeezy** | 5% + 50¢ per sale | Lower fees, EU tax handling |
| **Payhip** | 5% per sale (free plan) | Lowest fees |
| **Shopify** | $39/mo + 2.9% + 30¢ | If you want a full storefront |

**Recommendation:** Start with Gumroad for speed and simplicity. You can migrate later if volume justifies lower-fee platforms.

## File Preparation Checklist

Before uploading to Gumroad, ensure:

- [ ] Main guide PDF is properly formatted with table of contents
- [ ] All links in PDFs work (especially AI prompt formatting)
- [ ] Template files are provided in both PDF (for reading) and editable format (Markdown or DOCX)
- [ ] All files are named clearly (no "final_v2_FINAL.pdf")
- [ ] ZIP file tested — extracts cleanly on Mac and Windows
- [ ] Total file size is under Gumroad's limit (we're well under — mostly text)
- [ ] Product cover image is uploaded (1280x720px)
- [ ] Sales copy is pasted into product description
- [ ] Price is set to $39
- [ ] Launch discount code "LAUNCH" is created and tested
- [ ] Post-purchase email is configured
- [ ] Refund policy is set to 30 days
